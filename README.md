### Getting data from server

    step 3 : Get-ExecutionPolicy 
      
    step 4 : Get-ExecutionPolicy -list 
       
  -->step 2 , 3 and 4 may not required in some cases <br/>
        
    step 5: json-server --watch db.json --port 3031
